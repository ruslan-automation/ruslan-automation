## 🚀 Флагманские проекты (Мое портфолио)

Здесь собраны мои лучшие архитектурные решения. Каждый репозиторий отражает реальные бизнес-кейсы:

* 📞 **[ai-voice-sales-agent](https://github.com/ruslan-automation/ai-voice-sales-agent)**: Интеллектуальный голосовой ИИ-оператор для стоматологии (n8n + AmoCRM + Vapi + ElevenLabs). Звонит, общается голосом и заносит сделки.
* 📚 **[enterprise-rag-system](https://github.com/ruslan-automation/enterprise-rag-system)**: RAG-система для HR и юристов. Векторный поиск по корпоративным документам (Qdrant) с помощью локальных LLM (Groq, OpenRouter).
* 🏭 **[omnichannel-content-autopilot](https://github.com/ruslan-automation/omnichannel-content-autopilot)**: Полностью автоматизированный конвейер поиска, генерации и публикации контента на YouTube, Instagram и Telegram.
* 🎯 **[lead-radar-system](https://github.com/ruslan-automation/lead-radar-system)**: Система умного парсинга лидов (Custom JS внутри n8n) с бирж фриланса и Telegram с моментальной отправкой в воронку продаж.
* 🤖 **[careerboost-telegram-bot](https://github.com/ruslan-automation/careerboost-telegram-bot)**: Telegram-бот на Python (Aiogram 3, Asyncio, Pytest) с чистой архитектурой для поиска работы.
* 🧪 **[n8n-ai-agents-lab](https://github.com/ruslan-automation/n8n-ai-agents-lab)**: Лаборатория многоагентных систем. Содержит бота-дизайнера, квалификатора продаж и комплексного Email-ассистента на базе паттерна "Рой агентов" (Multi-Agent).
